# PETRÁN A PARTNERI - Advokátska kancelária

Moderná, responzívna webstránka pre advokátsku kanceláriu PETRÁN A PARTNERI v Prešove.

## 🌟 Funkcie

- **Responzívny dizajn** - Perfektné zobrazenie na mobile, tablete aj desktope
- **Moderné animácie** - Plynulé CSS animácie a scroll efekty
- **SEO optimalizované** - Meta tagy, Open Graph, sémantické HTML5
- **FAQ sekcia** - Často kladené otázky s accordion efektom
- **Google Mapa** - Interaktívna mapa s polohou kancelárie
- **Floating kontakt** - Plávajúce tlačidlo pre rýchle zavolanie
- **Cookie banner** - GDPR-compliant cookie consent
- **Preloader** - Elegantná loading animácia
- **Animované počítadlá** - Dynamicky načítavané štatistiky

## 🎨 Dizajn

- **Farebná schéma**: Tmavomodrá (#1a2744) + Zlatá (#c9a962)
- **Typografia**: Playfair Display (nadpisy) + Inter (text)
- **Štýl**: Elegantný, profesionálny, prémiový

## 📱 Sekcie

1. **Hero** - Hlavný banner s výzvou k akcii
2. **O nás** - História a predstavenie kancelárie
3. **Služby** - 4 oblasti práva (trestné, občianske, obchodné, daňové)
4. **Náš tím** - Prezentácia 3 advokátov
5. **Prečo my** - Výhody spolupráce
6. **FAQ** - Často kladené otázky
7. **Kontakt** - Kontaktný formulár a údaje
8. **Mapa** - Google Maps embed

## 🚀 Použitie

Jednoducho otvorte `index.html` v prehliadači. Žiadne závislosti nie sú potrebné.
